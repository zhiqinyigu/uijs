功能：
1.最小化、最大化、还原、关闭，功能按钮可设置显示或隐藏；
2.双击标题栏可以快速最大化和还原；
3.改变窗口大小，可设置是否启用该功能；
4.移动窗口，可设置是否启用该功能；
5.任务栏显示窗口列表，任务过多时支持翻页，且自动翻页至某激活窗口；
6.窗口图标，可设置是否显示图标；
7.任务栏图标，不设置则在普通任务栏显示；

参数：


	

		id
		每个窗口的独立ID，不可重复
	

	

		title
		窗口标题
	

	

		ico
		窗口图标地址，默认true
	

	

		move
		是否可以移动，默认true
	

	

		resize
		是否可以改变大小，默认true
	

	

		close
		是否可以关闭窗口，默认true
	

	

		min
		是否可以最小化，默认true
	

	

		max
		是否可以最大化，默认true
	

	

		task
		任务栏图标，不设置则在普通任务栏显示
	

	

		html
		静态内容
	

	

		url
		Iframe地址
	

	

		width
		宽度
	

	

		height
		高度
	

	

		top
		上边距
	

	

		left
		左边距
	



示例：
UI.dialogs.show({
	id : 'dialogs', //Id
	title : '功能说明', //标题
	ico : 'images/icons/desktop.png' , //图标
	move : true, //移动功能，默认true
	resize : true, //改变大小，默认true
	close : true, //关闭按钮，默认true
	min : true, //最小化按钮，默认true
	max : true, //最大化按钮，默认true
	//task : UI.G('settings'), //任务栏图标，不设置则在普通任务栏显示
	//html : '内容', //静态内容
	url : 'dialogs.html', //Iframe地址
	width : 500, //宽度
	height : 300, //高度
	top : 100, //上边距
	left : 200 //左边距
});
