样式应用
参数 说明
name(String) 菜单对象名
url(String) 对话框里面页面的url地址
size(String) 对话框的尺寸，默认提供三种尺寸，'small'、'medium'、'big'
width(Number) 对话框的宽度
height(Number) 对话框的高度度
close(Boolean) 默认值为ture，是否启用关闭对话框的功能
move(Boolean) 默认值为true，是否启用移动对话框的功能
resize(Boolean) 默认值为true，是否启用缩放对话框大小的功能
call(String) 关闭对话框时，执行某个JS命令

var dialog = new UI.Dialog({name:'dialog'});
dialog.show();
dialog.hide();
dialog.show({...});
